### 배열로 집합 만들기 
* 같은 자료형이 모인 집합은 배열로 표현할 수 있다. 
* 하지만 배열을 사용해 집합을 표현하려면 집합의 원소 개수와 배열의 
요소 개수가 항상 같아야 한다. 
* 따라서 집합을 표현하는 배열과, 이 배열의 정보를 담은 구조체를 함께
사용해야 한다. 
* 배열의 정보를 담은 구조체에는 집합의 최대 크기, 집합의 원소 개수, 
집합을 가리키는 포인터가 있다. 

```
typedef struct{
    int max; //집합의 최대 크기 
    int num; //집합의 원소 개수
    int * set;  //집합 
}IntSet; 

```
